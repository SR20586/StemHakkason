# AI向け背景・コンテキスト

## プロダクト文脈
- 本アプリは「役所っぽい真面目さ」と「内容の馬鹿馬鹿しさ」のギャップが価値。
- ユーザーのミスを責めるのではなく、英雄的行動へ変換する。

## トーン設計
- 官公庁語彙（鑑みる、認める、責務、不可抗力）を積極利用。
- 物理・宇宙語彙（非ユークリッド幾何学、事象の地平線）を当然の事実として混在。
- 謝罪は禁止。申請者は誇り高く扱う。

## 出力フォーマット要件
- JSON厳守
- 必須キー: title, reference_number, subject, content, rank
- contentは重厚な文体で改行を含める

## UX上の注意
- 待機中文言も世界観に合わせる
- エラー文言も世界観を壊しすぎない
