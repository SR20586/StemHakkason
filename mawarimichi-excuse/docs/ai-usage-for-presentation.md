# 発表用: AI活用説明メモ

## AIを活用したステップ
1. 画面構成案とUIトーン設計
2. APIルートとJSON契約の実装
3. エラーハンドリングと文言整備
4. ドキュメントテンプレート整備

## ヒューマンレビューとの分担
- AI: 叩き台作成と反復修正
- 人間: 要件整合、世界観の最終調整、提出品質担保

## 独自の工夫
- 官公庁文体 + 宇宙語彙のプロンプト融合
- 申請画面（ゴシック）と証明書（明朝）の意図的分離
- 角印演出と注釈多用でコンセプトを強化

## 他プロジェクト展開可能性
- 申請フォーム + AI生成 + 証明書化は他ドメインへ流用可能
- 例: 学習証明、旅程証明、イベント参加証明のジョーク化
